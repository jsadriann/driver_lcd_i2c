Construtor da classe DriverLCD_I2C.

Este construtor inicializa o display LCD I2C com o endereço especificado, número de colunas e linhas. Além disso, configura o cursor inicial e limpa o display após a inicialização.

Parameters

lcd_rowsNúmero de linhas do display LCD.

Exibe a medição de consumo de energia no LCD.

Este método exibe a medição de consumo de energia no LCD. A string "Consumption:" é seguida pela voltagem e corrente, com o respectivo símbolo de unidade. Se a mensagem não couber, o número de casas decimais é reduzido para 2.

Parameters
voltageA voltagem medida a ser exibida no LCD.
currentA corrente medida a ser exibida no LCD.
voltageUnitA unidade de voltagem a ser exibida.
currentUnitA unidade de corrente a ser exibida.
